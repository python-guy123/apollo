# Apollo OS

Welcome to Apollo OS! This is a fun, experimental "operating system" that runs entirely inside your web browser. 

It's built completely from scratch using basic web technologies: HTML, CSS, and JavaScript.

## What's Inside?

When you open Apollo OS, you'll get a full desktop environment that includes:

- **Desktop & Start Menu:** A classic taskbar, a dock, and desktop icons.
- **Window Manager:** You can drag, minimize, maximize, close, and even resize windows exactly like a real OS.
- **Settings App:** Easily change the color themes (like Cyberpunk or Apple Light Mode) and switch out your wallpapers.
- **The Terminal:** A fully working fake terminal where you can type commands like `ls`, `now`, and maybe even find a few hidden secrets!
- **Apps & Games:** It comes with a built-in web browser (Apollo Surf), an Apple-style Notes app, a Paint canvas, and a fully playable game of Pong.

## Terminal

  The Terminal features commands like `ls`, which lists the files in the code including images, but excluding the README.md, `now` which tells you the current date and time and more(included under the section Easter Eggs)

## Easter Eggs

Currently, the easter eggs are only in the Terminal app, but more will be added soon.
The Terminal app currently includes easter eggs such as:
- `ubuntu`: Shows an ASCII art logo of Ubuntu
- `arch`: Shows an ASCII art logo of Arch Linux
- `apollo`: Shows an ASCII art logo of Apollo OS
- `secret`: Shows a joke 'Secret Club' initiation message

## How to Use It

All you need to do is open the [Apollo OS Link](https://apollo-os.netlify.app/) file in any web browser (like Chrome, Safari, or Edge). Everything will boot up instantly.
